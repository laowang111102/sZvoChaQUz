# 前言

教学质量的评估是教育工作中至关重要的环节，它直接关系到教学效果的提升和教学方法的改进。基于微信小程序的教学质量评价系统SSM，旨在为广大师生提供一个便捷、高效的评价平台。本项目使用Java语言，结合Spring、Springmvc、MyBatis等框架，实现了一套完善的教学质量评价系统。现将项目源码托管至Gitee，欢迎各位开发者交流与学习。

# 内容介绍

本项目主要包括以下功能模块：学生评价、教师查询评价、管理员管理评价等。系统支持实时数据统计与分析，帮助教师了解自己的教学质量，同时也为学校管理者提供了有力的数据支撑。此外，基于微信小程序的访问方式，用户可以随时随地完成教学评价，大大提高了评价的便捷性和参与度。

# 技术介绍

- 语言：Java
- 使用框架：Spring、Springmvc、MyBatis，微信小程序

- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12、14、16
